Полезные инструменты для Godot Engine
-----------------------

* [https://github.com/Miziziziz/MizGodotTools](https://github.com/Miziziziz/MizGodotTools) - Полезные утилиты для игр от Miziziziz (создатель Endoparasitic)

* [https://github.com/Phazorknight/Cogito](https://github.com/Phazorknight/Cogito) - Шаблон для 3д иммерсив симов

* [https://github.com/Niekvdm/godot-plugins-gtml](https://github.com/Niekvdm/godot-plugins-gtml) - HTML в Godot для интерфейсов (не браузер)

* [https://github.com/MetalGanon/godot_ascii_toolkit](https://github.com/MetalGanon/godot_ascii_toolkit) - Для создания олдскульных интерфейсов

* [https://github.com/HungryProton/proton_control_animation](https://github.com/HungryProton/proton_control_animation) - Простые анимации для интерфейсов

* [https://github.com/stechyo/godot-steam-audio](https://github.com/stechyo/godot-steam-audio) - Продвинутый звук от Valve

* Шаблоны FPS контроллеров от Jeh3no:

	* [https://github.com/Jeh3no/Godot-Simple-State-Machine-First-Person-Controller](https://github.com/Jeh3no/Godot-Simple-State-Machine-First-Person-Controller) - Для простых контроллеров (Ходьба, прыжок, оглядется, бег, присядка)

	* [https://github.com/Jeh3no/Godot-Advanced-State-Machine-First-Person-Controller](https://github.com/Jeh3no/Godot-Advanced-State-Machine-First-Person-Controller) - Для сложных (Элементы паркура)

* [https://github.com/nndda/Theatre](https://github.com/nndda/Theatre) - Утилита для линейных диалогов
